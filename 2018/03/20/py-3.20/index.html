python初学之正则表达式

  今天晚上学习python，刷到了一个汇率转换的题目，需要提取字符串中的数字，在这里总结一下。

  • 方法一
      re.sub(pattern, repl, string, count=0, flags=0)
      在字符串 string 中找到匹配正则表达式 pattern 的所有子串，用另一个字符串 repl 进行替换。如果没有找到匹配 pattern 的串，则返回未被修改的 string。Repl 既可以是字符串也可以是一个函数。
    但是这种方法有个缺点，那就是当数字带有小数时，小数点会被删掉 = = 。
  • 方法二
      print re.findall(r"\d+\.?\d*",string)
      表达式：”\d+.?\d*“， \d+匹配1次或者多次数字，注意这里不要写成*，因为即便是小数，小数点之前也得有一个数字；.?这个是匹配小数点的，可能有，也可能没有；\d*这个是匹配小数点之后的数字的，所以是0个或者多个。
    但是这种方法好像是带有括号等等，不是怎么会用。
  •   对于此题目来说，需要删掉的是字母和字符，因为字符一定实在str[0]的位置，所以我直接从str[1]开始删，这样问题就解决了，但这只是对这个题目来说，实际上提取字符串中的数字并没有完全掌握。
